# Netflix Titles Visualization

This project visualizes the number of Netflix titles released each year using Python, Pandas, and Plotly. The visualizations are interactive, allowing you to explore trends in Netflix's content over time.

## Features
- Line chart showing the total number of titles released per year.
- Interactive hover to view exact year, number of titles, decade, and type (Movie or TV Show).
- Color-coded by decade for easier trend analysis.
- Different markers for Movies vs TV Shows.
- Fully interactive charts using Plotly.
